# Summer PEP Python Class Practice

This repository contains the Python practice work I completed during the Summer PEP class. It includes beginner-level exercises, daily notebook practice, small module-based programs, and simple project-style examples.

## Overview

This folder is mainly used for learning and revising:
- Python basics
- input and output
- variables and data types
- conditional statements and loops
- functions
- modules and packages
- file handling
- simple project structure organization

## Folder Structure

- Day 1 to Day 21: daily practice notebooks and exercises
- Assignment: assignment-related files
- Notes: study notes and learning references
- Pyhton-Pratice: practice programs and small Python examples

## Main Practice Areas

### 1. Daily Practice
The day-wise folders contain Jupyter notebooks used for practicing core Python concepts step by step.

### 2. Module Practice
The module_practice folder includes examples of:
- importing modules
- using basic calculator functions
- creating simple geometry-related functions
- building a small billing system

### 3. Python Practice
The Python_Practice folder includes examples of:
- calculator package structure
- ecommerce-style folder organization
- virtual environment setup

### 4. File Handling Examples
The Day 15 folder contains text files used for practicing reading and working with files.

## Learnings

Through this repository, I learned:
- how to write and run basic Python programs
- how to use variables, operators, loops, and conditions
- how to create and use functions
- how to organize code using modules and packages
- how to build small real-world examples like a billing system
- how to work with files and basic project folders
- how to set up and use a Python virtual environment

## Important Files

- Day 1/first.py: basic input and output example
- Pyhton-Pratice/module_practice/main.py: module import practice
- Pyhton-Pratice/module_practice/BillingSystem/main.py: simple billing example
- Pyhton-Pratice/Python_Practice/main.py: beginner calculator example

## How to Run

Make sure Python is installed on your system.

Example:

```bash
python "Day 1/first.py"
```

Billing example:

```bash
python "Pyhton-Pratice/module_practice/BillingSystem/main.py"
```

## Virtual Environment

A virtual environment folder is available at:

- Pyhton-Pratice/Python_Practice/myenv

To activate it on Windows:

```bash
Pyhton-Pratice\Python_Practice\myenv\Scripts\activate
```
